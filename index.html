About

This is an interative JavaScript implementation of domain coloring for complex functions. It's inspired by Hans Lundmark's page on the topic.

Expression syntax

          Expr := Sums
          Sums := Products "+" Sums
          | Products "-" Sums
          | Products
          Products := Power "*" Products
          | Power "/" Products
          | Power
          Power := Unary "^" Unary
          Unary := "-" Paren
          | "exp" Paren
          | "Log" Paren
          | Paren
          Paren := "(" Sums ")"
          | Value
          | Name
          Name := "a" | "b" | ... | "h" | "j" | ... | "z"
          Value := Number | Number "i"
          Number := Digits
          | Digits "." Digits

The color formula

Let z be the complex number and r,g,b the components of the color. frac(x) means the fractional part of x. 

          l = frac(log2 z),
          h = frac(arg z / 2π),
          r = l * min(2 * h, 1),
          g = l * h,
          b = 0
